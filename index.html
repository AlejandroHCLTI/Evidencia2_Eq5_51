JavaScript: Historia, Características y Conceptos

Historia de JavaScript

JavaScript fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape Communications. Originalmente se llamó Mocha, luego LiveScript y finalmente JavaScript. Su propósito era hacer que las páginas web fueran interactivas y dinámicas, algo que HTML por sí solo no podía lograr.

Creadores y evolución

- Creado por: Brendan Eich (Netscape).
- Primer lanzamiento: 1995.
- Estándar actual: ECMAScript (mantenido por ECMA International).
- Actualmente es uno de los lenguajes más utilizados en el desarrollo web, tanto en el lado del cliente (navegador) como en el servidor (Node.js).

Características principales

  • Lenguaje interpretado y de alto nivel.
  • Permite manipular el DOM (Document Object Model).
  • Soporta programación orientada a objetos, funcional y basada en eventos.
  • Es multiplataforma y se ejecuta en la mayoría de navegadores.
  • Permite la comunicación con servidores mediante AJAX o APIs REST.

Conceptos fundamentales

  • Variables: Espacios donde se almacenan datos (usando let, const o var).
  • Funciones: Bloques de código reutilizables que realizan una tarea específica.
  • Eventos: Acciones del usuario que desencadenan código (clics, teclas, etc.).
  • DOM: Representación estructurada del documento HTML que JavaScript puede manipular.
  • Objetos: Estructuras que agrupan datos y funciones relacionadas.

Formas de integrarse con HTML

JavaScript se puede integrar en una página web de tres formas:

  1. Dentro del archivo HTML usando la etiqueta <script>.
  2. En un archivo externo con extensión .js enlazado con <script src="archivo.js"></script>.
  3. En línea, dentro de atributos como onclick (no recomendado para proyectos grandes).
